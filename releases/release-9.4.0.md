# 9.4 - Voyage AI

TVault Intelligence is a different AI plugin for Obsidian. It goes beyond simple Q&A by introducing agents that maintain your vault's structure, retrieve information based on your explicit connections, and ground your knowledge in the real world.

This release includes support for [Voyage AI](https://www.voyageai.com/) as an embedding provider. This is a key step towards supporting Claude and others who do not bundle an embedding model with their main reasoning models. We now support:

-   Local embeddings, via Transformers.js and Ollama.
-   Cloud embeddings, via Gemini, Voyage AI, and Ollama.

*Install this update via BRAT using the repository link: [https://github.com/cybaea/obsidian-vault-intelligence](https://github.com/cybaea/obsidian-vault-intelligence)*
