---
description: how to safeguard and automate the upgrade of the transformers library
---

To upgrade the `@xenova/transformers` library and synchronized dependencies:

1. **Run Automation Script**: Execute the custom upgrade script.
// turbo
```bash
npm run upgrade-transformers
```

2. **Verify Output**: Check the console output for:
   - "SUCCESS: Transformers.js upgraded and verified."
   - Or "No update required."

3. **Commit Changes**: If changes were made, verify them and commit.
